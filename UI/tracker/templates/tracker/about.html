About FlipperBase

An automated penguin weight monitoring system designed to assist researchers in tracking African Penguin populations at Stony Point, Cape Town.

This project is developed by Group X as part of the EEE4113F course, aimed at supporting conservation efforts for the endangered African Penguin (Spheniscus demersus).

Project Objectives
  • Automate weight data collection of African Penguins
  • Provide researchers with reliable monitoring tools
  • Support behavioral pattern analysis during molting periods
  • Contribute to penguin conservation efforts
